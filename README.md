# Calculadora-No-Terminal-v1
Calculadora em Java executada no terminal. Projeto feito para revisar conceitos básicos da linguagem, como estruturas de decisão, repetição e entrada de dados. Esta é a primeira versão e a ideia é evoluir o código usando métodos conforme avanço nos estudos.

## O que foi praticado
- Estruturas de decisão (if / switch)
- Estruturas de repetição (while / do-while)
- Entrada de dados com Scanner

## Observações
Esta é a versão 1 do projeto. A ideia é evoluir o código futuramente, aplicando métodos e melhorias de organização conforme avanço nos estudos.
